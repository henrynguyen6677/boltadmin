⚡ BoltAdmin
The Lightning-Fast Open Source Admin Panel powered by PocketBase & AI.

BoltAdmin is a next-generation Content Management System (CMS) and Admin Panel. It combines the lightweight efficiency of the PocketBase core (Go + SQLite) with a modern, high-performance user interface and integrated AI assistants to streamline your data workflows.

✨ Key Features
🚀 Single Binary: Ship your entire Backend, Database, and UI as a single executable file. Zero-config deployment.

🎨 Modern UI (Inspired by Strapi): A smooth, intuitive administrative dashboard built for speed and productivity.

🤖 AI-First Engine: Built-in AI hooks for smart relation suggestions, content generation, and data validation.

🛠️ Dynamic Forms: Automatically generated management interfaces based on your database schema (JSON Schema).

🔌 Extensible Hook System: A powerful "pluggable" architecture to extend functionality using Go or JavaScript.

🔐 Built-in Auth: Ready-to-use user management, Role-Based Access Control (RBAC), and OAuth2 support.
